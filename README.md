# End-to-End Corporate HR Data Analytics Project

## 📌 Business Challenge
I audited and rehabilitated a raw, corrupted corporate employee database containing thousands of records, transforming it into an enterprise-ready interactive Power BI dashboard. The raw data was severely compromised by data-entry errors, structural column jams, and text-type alignment failures that prevented regional payroll tracking.

## 🛠️ Phase 1: Data Rehabilitation Workflow (Google Sheets)
I executed a targeted 4-stage data cleaning framework to restore data integrity:
1. **Numeric Financial Sanitization:** Stripped mathematical logging defects from phone records.
2. **Structural Column Splitting:** Parsed joined department and region tags into separate operational metrics.
3. **Data Type Calibration:** Converted text-trapped entries into proper numeric Currency and calendar Date dimensions.
4. **Systemic Blank Resolution:** Used isolated filters to identify and label missing age values with standard `N/A` corporate tags.

## 📊 Phase 2: Business Intelligence Architecture (Power BI)
To transform the audited master ledger into actionable executive insights, I engineered a multi-dimensional dashboard:
* **Executive KPI Blocks:** Developed precise values for Total Payroll Expenditure ($84.81M) and an exact metric for Total Active Workforce Headcount (1,020 employees).
* **Operational Analytics:** Structured a strategic grid layout tracking Department Headcounts, Regional Payroll Strains, and hybrid Remote Work Distribution models.
